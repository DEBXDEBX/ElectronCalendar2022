To add a Note to your Calender, You must first load or create a Year.

To create a Year

  1. Click on File and then select Create New Year
  2. Enter a number for the year and click add.
  3. Enter a name for the path and click save.

To load a Year

  1. Click on File and then select Load Year.
  2. Navigate to the file where you have your year file.
  3. Click on it and click open.

To add a Note

  1. Click a Month tab, and a note section will pop up.
  2. Click the small icon beside the word Note.
  3. This will bring up a form for your note. Enter your Note and click add.

To add a Image to a Note.

  1. To add an image to a Note, hold down alt and click on a note, the file picker will promt you for a path.
  2. Find your picture and select it and click open.
  3. To see the picture that belong's to the Note, Just click on the Note and the picture will pop up.
  4. To dismiss the picture just click the picture and it will go away.
  5. To remove an image path from a note dimiss the picture by clicking on it, it will disapear, and the hold down shift and click on the Note.
  6. To remove an image path that no longer exists, hold down shift and click on the Note.

To enter Edit and Delete Mode

  1. Click on Mode and select Edit and Delete Mode

To exit Edit and Delete Mode

  1. Click on Mode and select Read and Write

Delete a Note

  1. First enter Edit and Delete Mode.
  2. See above.
  3. To delete a note hold down the control key and click on the black trash can above the note.

To move a note up or down the screen

  1. First enter Edit and Delete Mode.
  2. See above.
  3. Click the arrow's beside the trash can.
  4. Click the up arrow to move up.
  5. Click the down arrow to move down.

To edit a note.

  1. First enter Edit and Delete Mode.
  2. See above.
  3. Click the pencil icon close the trash can.
  4. Edit Your Note in the form.
  5. Click Save Edit button.

To auto load Year files

  1. Click Settings and click Start Up Setting's Form.
  2. Check the box beside Auto Load Year's.
  3. Click the Add File To Auto Load List.
  4. Add as many file's as you like.
  5. After you add all your files, click the Save Settings button.
  6. To remove a file from the auto load list,
    enter Edit and Delete mode, hold down the control key and
    click the red trash can beside it and click the Save Settings button.

Start Up Settings

  1. Click Settings and click Start Up Setting's Form.
  2. Pick your theme.
  3. Pick your font.
  4. Add as many file's as you like, to the auto load list.
  5. After you make your selection's click the Save Settings button.

Factory Reset

  1. Click Settings and click Start Up Setting's Form.
  2. Click Factory reset
  3. This will set setting's back to default values
